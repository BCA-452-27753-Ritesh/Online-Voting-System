# 🗳️ Online Voting System

## 📌 Project Description

This is a secure Online Voting System developed using PHP, MySQL, HTML, CSS, and JavaScript.
It allows users to register, login, and vote securely.

---

## 🚀 Features

* 🔐 User Registration & Login
* 🗳️ Voting System
* 👨‍💼 Admin Panel
* 🔑 Token-based secure access
* 💳 Payment Integration (if added)

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL

---

## 📂 Project Setup

1. Download or Clone the repository
2. Move project to `htdocs` (XAMPP)
3. Import `database.sql` in phpMyAdmin
4. Run project in browser:

   ```
   http://localhost/online-voting-system
   ```
